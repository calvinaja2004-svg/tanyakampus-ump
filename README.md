# TANYAKAMPUS UMP

TanyaKampus UMP adalah aplikasi web forum tanya jawab sederhana berbasis PHP dan MySQL yang dibuat untuk membantu mahasiswa bertanya dan berdiskusi mengenai perkuliahan, materi, dan informasi kampus.

## Fitur
- Registrasi dan login user
- Posting pertanyaan
- Menjawab pertanyaan
- Edit dan hapus pertanyaan
- Sistem upvote jawaban
- Manajemen sesi login

## Teknologi
- PHP Native
- MySQL Database
- HTML & CSS
- Session Authentication

## Tujuan
Proyek ini dibuat sebagai implementasi pembelajaran pemrograman web dan sistem informasi berbasis database.

## Catatan
Project ini dijalankan menggunakan XAMPP atau web server yang mendukung PHP dan MySQL.
